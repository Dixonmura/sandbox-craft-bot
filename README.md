# SandBoxCraftBot

Учебный Telegram-бот с двумя режимами:
- Pomodoro-бот для тайм-менеджмента
- Movie Quiz по кадрам из фильмов

## Требования

- Java 17+
- Maven
- Токен Telegram-бота

## Запуск

1. Склонировать репозиторий.
2. Собрать проект:
   ```bash
   mvn clean package
Перед запуском приложения указать токен бота в переменной среды:

имя переменной: BOT_TOKEN

Запустить приложение:

bash
java -jar sandbox-multi-bot-2.0.1.jar

## Функциональность

- Режим Pomodoro: запуск рабочих сессий и перерывов с помощью Telegram-команд.
- Ведение сессии Pomodoro: бот отслеживает состояние таймера и напоминает о смене фаз.
- Режим Movie Quiz: показ кадра из фильма и варианты ответов.
- Подсчёт очков и присвоение ранга в кино-квизе по итогам игры.
